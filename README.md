# RPS – Rock Paper Scissors Game

A retro-themed Rock–Paper–Scissors game built with **Flask**, featuring:

- ✔ Classic mode (play vs CPU)
- ✔ Camera mode (play using your webcam + ML model)
- ✔ Multiplayer same-device mode (simultaneous key press RPS)

## How to Run the Project

Open your bash terminal

```bash
cd game
pip install -r requirements.txt
flask --app app run
```

then wait for it to fully load and when you see the
live host link, click it and you should be set.

## ✔️ Notes

- Make sure your webcam is connected before starting camera mode.
- If TensorFlow installation fails, ensure you're using **Python 3.10.11**.
